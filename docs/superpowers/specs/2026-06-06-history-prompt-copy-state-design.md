# 历史图片提示词已复制状态设计

## 背景

历史知识详情页中的图片提示词按钮目前在复制后短暂显示“已复制”，约 1.5 秒后恢复。用户无法区分当前页面会话中哪些图片提示词已经复制过。

## 设计

在 `HistoryPage` 内新增一个页面会话级的图片提示词已复制集合。图片提示词复制成功后，把对应按钮键加入集合，并持续显示“已复制”及已复制颜色。集合允许同时记录多个按钮，刷新页面后自然重置。

覆盖范围：

- 普通历史主题的每张卡片生图提示词。
- 朝代四件套中每个模块的每张卡片生图提示词。
- 封面生图提示词。

正文、JSON、封面文案等非图片提示词复制按钮继续使用当前约 1.5 秒的临时反馈。

选择另一条历史记录时清空图片提示词已复制集合，防止不同记录复用索引键后错误显示。

## 视觉

为已复制的图片提示词按钮增加 `history-copy-button--copied` 类。该状态使用强调色背景、强调色边框和对比清晰的文字颜色，并保留现有圆角与尺寸。

## 测试

更新历史页面测试，验证：

- 连续复制两个图片提示词后，两个按钮可同时保持已复制类和“已复制”文字。
- 图片提示词按钮使用持久状态，不因复制另一个提示词而清除前一个状态。
- 现有剪贴板写入行为保持不变。
