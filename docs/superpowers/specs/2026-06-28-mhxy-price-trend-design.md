# 梦幻西游物价趋势观察台设计

## 单一目标

物价记录页面只做一件事：让用户选择一个道具后，立即看懂它从首期到最新一期的价格走势。

## 信息架构

- 顶部为紧凑标题与“添加记录”折叠入口，新增表单默认关闭。
- 左侧观察列表按“来源 / 道具”聚合价格快照，每项只显示最新价、期数和首末涨跌。
- 右侧主视图只展示当前道具：最新价、首末涨跌、区间最低/最高、折线趋势。
- 图表下方仅列当前道具的历史快照，保留逐条删除能力。
- 无记录时给出明确空状态，引导从次要入口添加首条数据。

## 视觉方向

- 深海墨蓝 `#0C1727`：行情终端底色。
- 行情青 `#42D6B0`：当前选择和稳定上涨。
- 警示琥珀 `#F2B35D`：价格节点与重要数字。
- 数据灰 `#92A5BA`：刻度与辅助信息。
- 浅雾白 `#EDF7F4`：主文字。
- 标题使用 Bahnschrift Condensed，价格数字使用 Cascadia Mono，正文使用 Microsoft YaHei UI。
- 记忆点是一条带面积微光的“价格丝线”；不使用通用统计卡片墙。

## 响应式与无障碍

- 桌面端观察列表与趋势图左右布局，窄屏改为上下布局。
- 道具项使用原生按钮，提供清晰键盘焦点。
- 图表有可读的 `aria-label`，折叠添加入口使用原生 `details/summary`。
- 动效仅用于趋势线进入和按钮反馈，并尊重 `prefers-reduced-motion`。

## 测试

- 验证默认只聚焦一个道具及其历史。
- 验证切换道具后图表标题和历史同步更新。
- 验证新增表单位于默认关闭的次要入口。
- 保留价格快照提交、失败保留输入与删除测试。
